# 蓝晴-bot · QQ 开放平台配置清单

> 打开 [QQ 开放平台](https://bot.q.qq.com/) → 你的机器人 → **功能配置**  
> 以下字段与官网表单 **一一对应**，直接复制粘贴即可。

---

## 一、服务配置

**当前所有功能均为纯文字/图片指令，无需配置任何服务。跳过此页面。**
