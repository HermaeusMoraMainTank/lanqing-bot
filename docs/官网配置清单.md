# 蓝晴-bot · QQ 开放平台配置清单

> 打开 [QQ 开放平台](https://bot.q.qq.com/) → 你的机器人 → **功能配置**  
> 以下字段与官网表单 **一一对应**，直接复制粘贴即可。

---

## 说明

- **官网指令 = `/` 面板快捷入口**，上限 **24 条**；每个功能只登记 **一条主指令** 即可。
- **不登记也能用**：群聊 `@机器人 指令名` 仍可触发所有别名（如 `jrrp`、`抽小猪`、`午时已到`、`下班` 等）。
- **basic**（帮助 / ping / 你好）**不上架**，直接 `@机器人 帮助` 即可。

---
